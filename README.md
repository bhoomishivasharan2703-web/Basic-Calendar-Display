# Basic Calendar Display

## Description
This project is a GUI-based calendar application built using Python and Tkinter.

## Features
- Displays full year calendar
- Colorful UI
- User-friendly interface

## Technologies
- Python
- Tkinter
- Calendar Module

## How to Run
1. Install Python
2. Run the script:
   python calendar.py
